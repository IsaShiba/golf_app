# Golf Score Analysis App

自身のゴルフスコアを管理・分析するためのWebアプリケーションです。
Python (Streamlit) と Docker を使用して構築されています。

## 機能
- スコアデータの登録・編集・削除
- PostgreSQLデータベースによるデータ管理
- ストローク分析・パット数分析などの可視化

## 必要環境
- Docker
- Git
